# Final-Project

## Lighthouses of Taiwan – Historical Narrative Cartography

An interactive narrative map exploring Taiwan’s lighthouse network as maritime infrastructure, connecting lighthouse history with port activity, shipping routes, and fishing grounds.

The project visualizes how navigation aids, coastal ports, and maritime activity overlap spatially around Taiwan.

## Live demo:
https://eientk88.github.io/Lab5-Japan-Airport-Narrative-Cartography/

## Map Overview

The map combines multiple thematic layers:

Lighthouse locations (clustered markers)

Historical lighthouse eras

Qing (1874–1883)

Japanese Rule (1897–1939)

ROC (1949–1983)

Modern restoration (post-1945)

Historical port activity

Port-to-port shipping connections

Regional fishing grounds

Stylized glow particles and lighthouse beams

The goal is to illustrate Taiwan’s maritime system as a historical and spatial network, where lighthouses serve as time markers along shipping and fishing corridors.

## Technologies

Leaflet.js

Leaflet MarkerCluster

OpenStreetMap

Stadia Maps – Stamen Watercolor

CARTO Dark basemap

The interface uses a split layout with a narrative sidebar and an interactive map.

## Data Sources

Taiwan lighthouse locations (OpenStreetMap / GeoJSON)

Taiwan International Ports Corporation statistics

Historical lighthouse construction records

Generalized shipping corridors based on major Taiwanese port connections

Stylized fishing grounds representing major coastal fisheries zones

## Features

Dynamic day/night basemap switching

Historical era filtering for lighthouses

Searchable lighthouse database

Toggleable thematic layers

Stylized shipping routes and fishing grounds

Map summary panel showing visible features
